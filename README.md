# 지하주차장 좌표 메모 앱

신반포3차 경남아파트 지하2, 3, 4층 주차장 평면도 위에 원하는 위치를 직접 찍고 메모를 남길 수 있는 정적 웹앱입니다. 서버 없이 GitHub Pages 등으로 바로 호스팅할 수 있습니다.

## 사용 방법

1. 상단 탭에서 층(지하4층 / 지하3층 / 지하2층)을 선택합니다.
2. 모드를 선택합니다.
   - 보기: 지점을 탭하면 메모를 확인합니다.
   - 지점 추가: 도면을 탭한 자리에 새 지점이 추가되고, 이름/메모를 입력하는 창이 바로 뜹니다.
   - 메모 수정: 지점을 탭해서 이름과 메모를 고칩니다.
   - 지점 삭제: 지점을 탭하면 바로 삭제 확인이 뜹니다.
3. 검색창에 이름/번호/메모 내용을 입력하면 해당 지점으로 자동 스크롤되며 깜빡입니다.
4. 하단의 내보내기 버튼으로 현재까지 작업한 데이터를 JSON 파일로 저장할 수 있고, 불러오기로 다시 불러올 수 있습니다.

## 데이터 저장 방식

- 브라우저(localStorage)에 자동 저장되어, 같은 기기에서는 새로고침해도 유지됩니다.
- 다른 사람과 공유하거나 여러 기기에서 같은 데이터를 보려면, 내보내기한 JSON 파일을 `data/2f.json`, `data/3f.json`, `data/4f.json`으로 덮어쓰고 깃에 커밋/푸시하면 됩니다. 그러면 다음에 접속하는 모든 사람이 그 데이터를 기본값으로 보게 됩니다(단, 로컬에 이미 저장된 기기는 로컬 데이터가 우선됩니다. 최신 공유 데이터를 받으려면 브라우저 저장공간을 지우거나 불러오기로 직접 덮어써주세요).

## 폴더 구조

```
index.html       메인 화면
style.css        스타일
app.js           동작 로직
images/2f.jpg    지하2층 도면
images/3f.jpg    지하3층 도면
images/4f.jpg    지하4층 도면
data/2f.json     지하2층 지점 데이터
data/3f.json     지하3층 지점 데이터
data/4f.json     지하4층 지점 데이터
```

## GitHub Pages로 배포하기

1. 이 폴더 전체를 깃 저장소에 푸시합니다.
2. 저장소 Settings > Pages 에서 배포할 브랜치와 폴더(루트)를 지정합니다.
3. 발급된 주소로 모바일/PC 어디서든 접속해 사용할 수 있습니다.

## 모바일 사용

별도 설치 없이 모바일 브라우저로 접속하면 바로 사용됩니다. 화면 하단 "홈 화면에 추가" 기능을 쓰면 앱처럼 아이콘으로 추가할 수 있습니다.
